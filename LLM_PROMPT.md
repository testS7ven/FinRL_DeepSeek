# LLM Review Instructions

To comply with the submission requirements, you must obtain 4 public conversation links from ChatGPT, Claude, Grok, and Gemini.

## The Prompt to Use

Paste the following into each chatbot (replacing `[PASTE LATEX CONTENT]` with the content of `paper.tex`):

```text
[PASTE LATEX CONTENT HERE]

Here is my draft. Is it ready for submission to NeurIPS 2026, or should I revise it before, and how? Write a NeurIPS-level review of my draft, based on NeurIPS evaluation criteria and reviewer guidelines:
https://neurips.cc/Conferences/2015/PaperInformation/EvaluationCriteria
https://neurips.cc/Conferences/2025/ReviewerGuidelines
```

## Checklist for Review Links

1. [ ] **ChatGPT**: [Share Link](https://help.openai.com/en/articles/7925741-chatgpt-shared-links-faq)
2. [ ] **Claude**: [Share Link](https://support.claude.com/en/articles/10593882-sharing-and-unsharing-chats)
3. [ ] **Grok**: [Share Link](https://x.ai/legal/faq#can-i-share-my-grok-conversations)
4. [ ] **Gemini**: [Share Link](https://docs.cloud.google.com/gemini/enterprise/docs/share-conversations)

> [!IMPORTANT]
> If any of the chatbots does not recommend submission, improve the draft and iterate until you get a positive recommendation.
