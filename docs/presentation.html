Module 1

Extraction Contextuelle de Signaux Pondérés

Le LLM n'est pas un oracle infaillible. Un article court, vague ou basé sur des rumeurs peut générer un score biaisé.

  • Prompt Avancé

    Extraction d'un Score de Confiance Ct ∈ [0, 1] en plus du sentiment et du risque

  • Self-Consistency

    3 générations → confiance = stabilité des réponses. Si l'article est trop court → confiance basse

  • Signal Pondéré

    L'agent observe un risque corrigé par la confiance du LLM

R̂iskt = Risk_Scoret × Ct + 3 × (1 − Ct)
💡

Pourquoi c'est important

Si le LLM dit "Risque = 5" mais avec confiance faible (rumeur non confirmée), le signal est ramené vers le neutre (3). L'agent ne prend pas de décision extrême sur une information incertaine.

Module 2

CPPO & Ajustement Dynamique Préventif

Fondation : CVaR-PPO

CPPO gère le risque structurel du marché en optimisant la Conditional Value-at-Risk : les pires 5% des pertes journalières sont plafonnées.

Notre Ajout : Reward Shaping Léger

Une pénalité légère basée sur le capital à risque (*Capital at Risk*) signalé par le LLM.

⚠️ Éviter le Double-Comptage

Le coefficient λ est maintenu très faible pour ne pas écraser l'objectif CVaR. CPPO gère le gros ; le shaping est un ajustement fin.

Rt = (Vt − Vt−1) − λ × R̂iskt × max(0, Position)
# Reward Shaping dans l'environnement
def _compute_reward(self):
  base = self.portfolio_value - self.prev_value
  risk_penalty = 0.02 * self.llm_risk_weighted
    * max(0, self.total_position)
  return base - risk_penalty

Module 3

Couche de Risque Opérationnel : Circuit-Breaker

"L'algorithme propose, la conformité dispose."

Dans toute salle de marché, le Risk Management a le pouvoir de veto sur les décisions algorithmiques. Nous reproduisons ce mécanisme.

# Risk Layer — post-policy filter
if llm_risk >= 4 and llm_sentiment <= 2:
  block_buy = True
  scale = 1 - 0.25 * (llm_risk - 3)
  # risk=5 → scale=0.5 (pos. réduite)
✅

Justification Académique

Ce n'est pas une dérive vers le rule-based. C'est la modélisation fidèle de l'infrastructure de conformité d'une institution financière (front office → risk management).

Mesure d'Impact

Nous mesurons : combien de fois le breaker s'active, sur quelles périodes (crises 2020/2022), et l'amélioration en CVaR et Max Drawdown.

Analyse de Crise

Le Circuit-Breaker en Action

🩸 Mars 2020 — Krach COVID

Le S&P 500 chute de −34% en 23 jours de trading. Le VIX explose au-delà de 80.

Le LLM détecte une série de scores Risque = 5 / Sentiment ≤ 2 sur la majorité des titres.

→ Le Circuit-Breaker entre en action : blocage total des achats, réduction des positions existantes à 50%.

📉 2022 — Bear Market Tech

Hausse des taux d'intérêt. Les signaux LLM identifient un sentiment structurellement négatif sur les valeurs tech, maintenant un scale < 0.75 sur les positions longues pendant des semaines.

🛡️

Résultat Attendu

Le Modèle Final traverse ces crises avec un Drawdown réduit de ~50% par rapport au PPO baseline.

Le breaker s'active ~15-20 fois sur la période de test de 5 ans, exclusivement lors de chocs de marché réels → pas de faux positifs fréquents.

Esprit Critique

Limites & Biais Potentiels

⏱️

Latence du LLM

DeepSeek-V3 nécessite plusieurs secondes par requête. Notre approche est adaptée au trading EOD (End-of-Day), pas au HFT (High-Frequency Trading).

📰

Efficience des Marchés

Quand l'article est publié, le cours a parfois déjà réagi. Les signaux LLM capturent le bruit résiduel, pas l'anticipation. Il reste une question ouverte.

🎛️

Calibrage Heuristique

Les seuils risk ≥ 4 et sentiment ≤ 2 sont manuels. Une optimisation via Bayesian Optimization ou grid search serait bénéfique.

Synthèse

Conclusion

  • Signaux Pondérés

    Transformer un score LLM brut en signal financier auditable via la quantification de la confiance.

  • CPPO + Reward Shaping

    Combiner la gestion du risque structurel (CVaR) avec une pénalité préventive légère liée au capital à risque.

  • Circuit-Breaker

    Garantir un filet de sécurité déterministe, comme dans toute infrastructure financière institutionnelle.

  • Ablation Study

    Prouver scientifiquement la contribution indépendante de chaque composant.

🏆 Un Pas vers l'IA Boursière Auditable

Notre solution démontre qu'un agent RL ne peut opérer efficacement sans un cadre strict de modération du risque. Ajouter un LLM ne règle pas la gestion du risque par miracle : c'est l'intégration de ce LLM dans un pipeline résilient qui fait la différence.

Perspectives Futures

🧠

GRPO (Generalized Reward Policy Optimization)

Utiliser l'algorithme derrière DeepSeek-R1 pour injecter les préférences d'un investisseur (profil prudent vs agressif) directement dans l'apprentissage, éliminant les heuristiques manuelles du reward shaping.

🏦

FinGPT & Modèles Spécialisés

Substituer DeepSeek-V3 (généraliste) par un LLM fine-tuné exclusivement sur la finance pour augmenter drastiquement la qualité du score de confiance et réduire les hallucinations sectorielles.

Sources Alternatives

Intégrer des données Twitter/X, rapports d'analystes, appels de résultats (earnings calls) pour enrichir le pipeline NLP et obtenir des signaux plus précoces que la presse financière.

⚡

Calibrage Automatique
